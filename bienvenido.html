Acerca de esta página

Esta es una página web básica creada con HTML y CSS. El objetivo es mostrarte cómo estructurar una página web sencilla y cómo darle estilo con CSS.

CSS (Cascading Style Sheets) es el lenguaje utilizado para definir la apariencia y el diseño de una página web. En este ejemplo, hemos utilizado CSS para dar color, márgenes y otros estilos a los elementos de la página.
